Anlisis de Eficacia y Coste Goleador Temporada 2024-25

Bienvenido/a. Este cuaderno explora el rendimiento ofensivo y la rentabilidad salarial de 11 atacantes destacados de la liga 2024-25.
A lo largo del anlisis responderemos a tres preguntas clave:

  1. Quin genera ms goles y con qu frecuencia?

    • Goles totales
    • Goles / Partido
    • Goles / Tiro

  2. Cun certeros son sus remates?

    • Goles / Partido
    • Goles / tiro

  3. Cunto cuesta cada gol o disparo?

    • Euros / Gol
    • Euros / Tiro
    • Euros / Tiro a Puerta

Variables del dataset

Volumen de juego

  • Partidos | Encuentros disputados
  • Minutos | Minutos totales sobre el csped

Produccin ofensiva

  • Goles | Goles marcados en la temporada
  • Tiros | Disparos totales intentados
  • Tiros a Puerta | Disparos que fueron a portera

Economa

  • Salarios () | Salario anual estimado

Estas mtricas permiten comparar eficiencia goleadora y rentabilidad econmica de estos 11 futbolistas destacados de esta temporada, y constituyen el hilo conductor de los grficos y tablas posteriores.

Los datos han sido recopilados de las siguientes pginas:

def interpretar(ratio):
    if ratio >= 0.8:
        return "Marca prcticamente cada partido. Nivel lite."
    elif ratio >= 0.6:
        return "1 gol cada 1-1.6 partidos. Ratio alto."
    elif ratio >= 0.4:

    elif ratio >= 0.3:
        return "1 gol cada 3 partidos. Ratio medio."
    else:
        return "1 gol cada 4+ partidos. Aporta ms en creacin."

eficiencia["Conclusin"] = eficiencia["G/P"].apply(interpretar)
