Análisis de Eficacia y Coste Goleador — Temporada 2024-25¶

Bienvenido/a. Este cuaderno explora el rendimiento ofensivo y la rentabilidad salarial de 11 atacantes destacados de la liga 2024-25.
A lo largo del análisis responderemos a tres preguntas clave:

  1. ¿Quién genera más goles y con qué frecuencia?

    • Goles totales
    • Goles / Partido
    • Goles / Tiro

  2. ¿Cuán certeros son sus remates?

    • Goles / Partido
    • Goles / tiro

  3. ¿Cuánto cuesta cada gol o disparo?

    • Euros / Gol
    • Euros / Tiro
    • Euros / Tiro a Puerta

Variables del dataset¶

Volumen de juego

  • Partidos | Encuentros disputados
  • Minutos | Minutos totales sobre el césped

Producción ofensiva

  • Goles | Goles marcados en la temporada
  • Tiros | Disparos totales intentados
  • Tiros a Puerta | Disparos que fueron a portería

Economía

  • Salarios (€) | Salario anual estimado

Estas métricas permiten comparar eficiencia goleadora y rentabilidad económica de estos 11 futbolistas destacados de esta temporada, y constituyen el hilo conductor de los gráficos y tablas posteriores.

Los datos han sido recopilados de las siguientes páginas:

  • Transferkmark
  • Capology
  • Fotmob

# Clasificación textual
def interpretar(ratio):
    if ratio >= 0.8:
        return "Marca prácticamente cada partido. Nivel élite."
    elif ratio >= 0.6:
        return "≈1 gol cada 1-1.6 partidos. Ratio alto."
    elif ratio >= 0.4:
        return "≈1 gol cada 2 partidos. Rendimiento sólido."
    elif ratio >= 0.3:
        return "≈1 gol cada 3 partidos. Ratio medio."
    else:
        return "≈1 gol cada 4+ partidos. Aporta más en creación."

eficiencia["Conclusión"] = eficiencia["G/P"].apply(interpretar)

# Ordenar de mayor a menor eficiencia
eficiencia = eficiencia.sort_values("G/P", ascending=False)
